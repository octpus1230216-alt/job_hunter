# 使用说明 — 半自动求职工具

## 概述

本工具帮助你发现职位、智能匹配简历、定制求职材料。

**工作流程：** AI 分析简历 → 生成搜索关键词 → 多渠道搜索职位 → AI 匹配打分 → 生成定制简历

---

## 环境准备（一次性）

### 1. 安装 Python

确保已安装 Python 3.10+。项目中已自带 Python 3.13.12。

### 2. 安装依赖

打开终端，执行：

```cmd
cd "C:\ProgramData\WorkBuddy\chromium-env\6od91a\WorkBuddy\2026-06-08-10-32-07\job-hunter"
C:\ProgramData\WorkBuddy\chromium-env\6od91a\.workbuddy\binaries\python\envs\default\Scripts\pip.exe install -r requirements.txt
```

### 3. 配置 API Key

编辑 `config.yaml`，填入 API Key：

```yaml
llm:
  provider: deepseek          # deepseek / openai / ollama
  api_key: "sk-xxxxxxxxxxxx"  # 你的 API Key
  model: deepseek-chat
```

支持三种后端：
- **DeepSeek**：`api_key` + `model: deepseek-chat`（推荐，便宜好用）
- **OpenAI**：`api_key` + `model: gpt-4o`
- **Ollama**：本地运行，无需 API Key，`model: llama3`

## 使用流程

### Step 1：上传简历 + AI 职业定位

1. 打开 `http://localhost:8501` → 点击左侧「⚙️ 配置」
2. 上传简历（支持 PDF / DOCX）
3. AI 自动解析简历内容
4. 切换到「🔍 发现职位」→ 点击「🚀 开始 AI 职业定位」
5. AI 分析你的能力，生成：
   - 核心求职方向（策略A）
   - 可迁移方向（策略B，跨行业求职用）
   - 中英双语搜索关键词
   - 弱项提醒 + 薪资锚定

### Step 2：搜索职位

#### 海外平台（自动）
1. 在「发现职位」→ 「🌍 海外平台自动搜索」标签页
2. 选择目标国家和平台（Indeed/LinkedIn/GlassDoor）
3. 点击「前台搜索」或「后台搜索」

#### 国内平台（Boss直聘）
1. 在「发现职位」→ 「🏠 国内平台直达搜索」标签页
2. 确保看到 `✅ 采集器运行中` 和 `✅ CDP Chrome 就绪`
3. **点击关键词旁的「🔍 搜索」按钮**
4. CDP Chrome **自动跳转到 Boss直聘 搜索结果**
5. 切换到 Chrome 窗口，浏览职位列表
6. **点击感兴趣的职位卡片**查看详情 → 详情自动被补全
7. 回到 Streamlit，刷新页面 → 看到 🟢 完整职位 → **点击「导入并匹配」**
8. AI 自动打分 + 标注行业类型

### Step 3：审核挑选

在「📊 审核挑选」页面：
- 按匹配度排序
- 按行业标签筛选（核心 / 可迁移 / 探索）
- 勾选想要投递的岗位
- 点击「确认选择」

### Step 4：生成定制简历

在「✨ 生成简历」页面：
- 选择岗位
- AI 分析公司风格
- 生成中英双语定制简历 / 求职信
- 一键下载

## 常见问题

### Q: Chrome 启动失败？

**A:** 关闭所有 Chrome 窗口后重试。如果还不行，手动运行：
```powershell
Start-Process -FilePath "C:\Program Files\Google\Chrome\Application\chrome.exe" -ArgumentList "--remote-debugging-port=9222 --user-data-dir=C:\temp\chrome-debug"
```

### Q: Boss直聘页面空白/回退？

**A:** 说明反检测补丁没生效。确认脚本输出中有 `✅ Chrome 已启动（反检测已激活）`。

### Q: 点击搜索按钮 Chrome 没反应？

**A:** 确认 CDP 脚本正在运行（终端 B 中有 `🟢 监听中...`）。确认 Streamlit 中显示 `✅ CDP Chrome 就绪`。

### Q: 采集到职位但只有基本信息（🟡）？

**A:** 正常现象。Boss直聘的设计是在点击职位卡片后才加载详细 JD。在 CDP Chrome 中点击职位 → 系统自动补全 → 状态变为 🟢 完整。

### Q: API Key 报错？

**A:** 检查 `config.yaml` 中 `api_key` 是否正确。DeepSeek 需要充值才能使用（余额 > 0）。

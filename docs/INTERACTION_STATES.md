# 交互状态文档 — 半自动找工作工具

## 1. 全局状态定义

### 1.1 Session State 变量
| 变量名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `resume_parsed` | dict/None | None | 解析后的简历数据 |
| `resume_path` | str/None | None | 简历文件路径 |
| `llm_client` | LLMClient/None | None | LLM 客户端实例 |
| `config` | dict | {} | 用户配置（从 config.yaml 加载） |
| `career_report` | dict/None | None | AI 职业定位报告 |
| `all_jobs` | list | [] | 所有待审核的岗位 |
| `jobs_df` | DataFrame/None | None | 海外搜索结果 DataFrame |
| `selected_jobs` | list | [] | 用户选择的岗位（传给生成页） |
| `selected_indices` | list | [] | 审核页中勾选的索引 |
| `manual_jobs` | list | [] | 手动粘贴的岗位 |
| `gen_{i}` | dict/None | None | 第 i 个岗位的生成结果 |

### 1.2 持久化状态
| 文件 | 说明 | 读写时机 |
|------|------|---------|
| `data/resume_parsed.json` | 简历解析结果 | 解析时写，启动时读 |
| `config.yaml` | 用户配置 | 配置页保存时写，启动时读 |
| `data/career_report_*.json` | 职业定位报告 | 定位完成时写，发现页加载时读 |
| `data/applications/applications.json` | 投递记录 | 添加/更新时写，追踪页加载时读 |
| `data/search_tasks/*.json` | 后台任务 | 提交/进度更新/完成时写 |
| `data/daily_digests/digest_*.json` | 每日精选 | 每日运行 daily_digest.py 时写 |
| `data/domestic_jobs/current_session.json` | 浏览器采集数据 | collector_server 收到数据时写 |

## 2. 页面交互状态

### 2.1 首页 (app.py)

#### 状态：初始化
- **触发**：用户首次打开
- **行为**：
  - 从 `config.yaml` 加载配置
  - 检测 `data/resume_parsed.json` 是否存在，存在则自动加载
  - 初始化 `llm_client`（缓存）
- **UI**：显示欢迎页，指标卡片，快速开始步骤

#### 状态：侧边栏
- **常驻显示**：
  - Streamlit 原生页面导航
  - 运行状态指示器：简历状态、AI 状态、待审核岗位数
  - 数据本地化提示

### 2.2 配置页 (01)

#### 状态：简历区域
| 子状态 | 触发 | UI | 行为 |
|--------|------|----|------|
| 空状态 | 未上传简历 | 文件上传组件 | 等待用户上传 |
| 已保存 | 文件上传完成 | 文件名 + 成功提示 | 显示解析按钮 |
| 解析中 | 点击解析按钮 | `st.status` + 分步文字 | API 调用 |
| 解析成功 | 解析完成 | 简历预览卡片 | 保存到 session_state + 磁盘 |
| 解析失败 | 解析异常 | 错误提示 + 恢复建议 | 提示尝试纯文本模式 |

#### 状态：API 设置
| 子状态 | 触发 | UI | 行为 |
|--------|------|----|------|
| 未配置 | API Key 为空 | 输入框 + 平台选择 | 等待输入 |
| 测试中 | 点击保存并测试 | spinner | API 调用 |
| 连接成功 | 测试返回200 | 成功提示 + 响应预览 | 更新 session_state |
| 连接失败 | 测试异常 | 警告提示 | 保存配置但不更新 client |

#### 状态：求职偏好
| 子状态 | 触发 | UI | 行为 |
|--------|------|----|------|
| 编辑中 | 页面加载 | 文本输入框 + 滑块 + 复选框 | 显示当前配置值 |
| 已保存 | 点击保存按钮 | 成功提示 | 写入 config.yaml + 更新 session_state |

| 后台搜索中 | 点后台搜索 | 任务 ID + 提示 | 页面可自由浏览 |

| 搜索取消 | 用户点停止 | 取消提示 | 任务标记为 cancelled |

#### 状态：后台任务面板
| 子状态 | 任务状态 | 图标 | 操作按钮 |
|--------|---------|------|---------|
| 运行中 | running | 🔄 | ⏹ 停止 |

#### 状态：审核挑选

### 2.4 生成页 (04)

#### 状态：前置条件
| 子状态 | 条件 | UI | 行为 |
|--------|------|----|------|
| 无简历 | `resume_parsed` is None | 引导提示 | 提示去配置页 |
| 无岗位 | `selected_jobs` 为空 | 手动输入表单 | 可手动添加岗位 |
| 就绪 | 简历 + 岗位都有 | 岗位列表 | 显示生成按钮 |

#### 状态：生成操作
| 子状态 | 触发 | UI | 行为 |
|--------|------|----|------|
| 未生成 | 岗位展开 | 岗位信息 + 生成按钮 | 等待点击 |
| 批量生成中 | 点击一键生成 | 进度条 + 逐条文字 | LLM 调用 |
| 单个生成中 | 点击单个生成 | `st.status` + 分步文字 | LLM 调用 |
| 生成完成 | API 返回 | 结果预览 + 下载按钮 | 保存到 session_state + 文件 |
| 生成失败 | API 异常 | 错误提示 | 提示检查 API |

#### 状态：已生成结果
| 子状态 | 条件 | UI | 行为 |
|--------|------|----|------|
| 简历预览 | `gen_{i}` 存在 | 折叠面板：摘要 + 技能 | 可展开查看 |
| 求职信预览 | `gen_{i}` 存在 | 折叠面板：主题 + 正文 | 可展开查看 |
| 速查卡 | `gen_{i}.quick_card` 存在 | 折叠面板：5个子项 | 可直接复制 |
| 下载简历 | 文件已保存 | 下载按钮 | 文件保存为 HTML |
| 下载求职信 | 文件已保存 | 下载按钮 | 文件保存为 MD |
| 重新生成 | 点击重新生成 | 重新调用 LLM | 覆盖旧结果 |

#### 状态：下载按钮生命周期（已修复）
- 生成结果存入 `session_state.gen_{i}`
- 下载按钮在按钮回调**外部**渲染（从 session_state 读取）
- 用户下载后不会触发 rerun 导致按钮消失

## 3. 错误状态

### 3.1 AI API 错误
| 错误 | 触发条件 | 用户可见 | 恢复方式 |
|------|---------|---------|---------|
| API Key 无效 | 401 响应 | ❌ 测试连接失败 | 重新输入正确 Key |
| 超时 | 请求超 300 秒 | ❌ 操作失败提示 | 重试或切换 provider |
| 速率限制 | 429 响应 | ❌ 操作失败提示 | 等待后重试 |
| 模型不可用 | 404 响应 | ❌ 连接失败提示 | 检查模型名是否正确 |

### 3.2 搜索错误
| 错误 | 触发条件 | 用户可见 | 恢复方式 |
|--------|---------|---------|---------|
| 国家码无效 | JobSpy 不支持 | ⚠️ 跳过提示 | 选择支持的国家 |
| 平台不可用 | API 异常 | ⚠️ 跳过提示 | 切换平台 |
| 零结果 | 关键词无匹配 | ⚠️ 空结果提示 | 更换关键词 |
| 依赖缺失 | import 失败 | ❌ 错误提示 | 安装依赖 |

### 3.3 简历解析错误
| 错误 | 触发条件 | 用户可见 | 恢复方式 |
|--------|---------|---------|---------|
| PDF 扫描版 | 文字提取为空 | ❌ 解析失败 | 尝试纯文本模式 |
| 文件损坏 | 无法打开 | ❌ 解析失败 | 更换文件 |
| AI 超时 | LLM 调用超时 | ❌ 解析失败 | 尝试纯文本模式 |

## 5. 响应式行为

| 操作 | 响应时间预期 | 进度指示 | 是否阻塞 |
|------|-------------|---------|---------|
| 简历解析（文本） | < 2 秒 | spinner | 否（st.rerun） |
| 简历解析（AI） | 10-30 秒 | st.status + 分步 | 是 |
| AI 职业定位 | 30-90 秒 | st.status + 分步 | 是 |
| 海外前台搜索 | 30-120 秒 | st.status + 逐条 | 是 |
| 海外后台搜索 | 不限 | 任务面板轮询 | 否 |
| AI 全网搜索 | 30-90 秒 | st.status + 逐条 | 是 |
| 单个岗位匹配 | 5-15 秒 | spinner | 是 |
| 单个简历生成 | 15-45 秒 | st.status + 分步 | 是 |
| 批量生成 | 按岗位数 × 30 秒 | 进度条 | 是 |
| 投递记录保存 | < 1 秒 | 无 | 否 |
